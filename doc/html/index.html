p5vlc

p5vlc provides interface for Perl developers to manipulate VideoLAN Client via it's Telnet interface.

Along with interface to VideoLAN Client itself, p5vlc provides object-oriented interface to construct command line (configuration file and VLM interface options) arguments. Take a look at VLC::Helpers modules.

Documentation and examples

There is documentation that was extracted from POD comments from source files. I'm sorry, but it's badly cross-referenced. Here are links to most interesting places:

  • VLC - main entry point
  • VLC::Broadcast - broadcast media type
  • VLC::Schedule - managing schedules

Documentation on helpers is also avaliable. Stream manipulation modules lack method descriptions -- please look at VideoLAN Howto on streaming about option names and their values.

  • VLC::Helper::Stream::Standard - creating std stream type definitions

There are two usage examples in the release archive: one for Telnet interface itself, another -- using Helper classes.
